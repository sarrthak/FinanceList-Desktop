FinanceList-Desktop

FinanceList is a simple finance managing tool. For more information on how to use this tool, take a look at the wiki.

Features

FinanceList allows you to keep track of your finances with ease. Some of the included features are:

  • Filter data by any parameter
  • Add recurring transactions which get executed automatically in a given time interval
  • Allocate sums automatically between different budgets
  • Export a monthly overview as a .pdf-file
  • And many more! Download FinanceList and try them out!

For more information about this application, visit the FinanceList wiki. You can request new features at any time here.

Installation

You can get the latest version of FinanceList here. Simply download the correct ZIP-archive and unzip it.
